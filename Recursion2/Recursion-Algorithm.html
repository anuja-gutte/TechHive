Recursive Algorithm for Factorial Calculation


Problem Statement: Given a non-negative integer n, calculate the factorial of n denoted as n!. The factorial of n is the product of all positive integers less than or equal to n.

Step-by-Step Algorithm:

Step 1:   Start.

Step 2:   Input: Take an integer n as input.

Step 3:   Base Case Check:

            If n=0, return 1. (This is because 0!=1).

Step 4:   Recursive Case:

            If n>0, return n*factorial(n-1).

            else Call the function recursively with n-1 until the base case n=0 is reached.

Step 5:   End.

Pseudocode

                
                    
    function Factorial {
        
       
            // Base case
            if (n == 0) {
                return 1;
            }
            else
            // Recursive case
            return n * factorial(n - 1);
        }

Code Example


    // This is a recursive function to calculate factorial
    public class Factorial {
        
        public static int factorial(int n) {
            // Base case
            if (n == 0) {
                return 1;
            }
            // Recursive case
            return n * factorial(n - 1);
        }
    }

Analysis


  1. The time complexity of recursion depends on several factors, including the nature of the recursion (how many recursive calls are made and how the problem is divided).
  2. In the recursive approach of finding a factorial, for each value of n, we make a recursive call to compute the factorial of (n-1) This process continues until we reach the base case (when n=0).
  3. For n, there is 1 recursive call to comput factorial(n-1).
  4. For n-1, there is 1 recursive call to compute factorial(n-2).
  5. This continues until the base case is reached.
  6. Thus,there are n n recursive calls in total, each performing a constant amount of work (a multiplication and a function call).
  7. Therefore the time complexity is: O(n)
