Code Implementation: Recursion in Java


How to implement Recursion in Java

Recursion in Java is implemented by having a method call itself to solve smaller instances of a problem. A recursive function typically has two main parts:

1.  Base Case: This stops the recursion when the problem is simple enough to be solved directly.

2.  Recursive Case: This breaks the problem down into smaller instances of the same problem and calls itself with those smaller instances.



Reversing a String using Recursion

    class StringReverse { 

    //recursive method to reverse a given string

    void reverseString(String str) 
    { 
        //base condition
        if ((str==null)||(str.length() <= 1)) 
           System.out.println(str); 
        else
        { 
            System.out.print(str.charAt(str.length()-1)); 
            reverseString(str.substring(0,str.length()-1)); 
        } 
    }
}
    public class Main{
        public static void main(String[] args)  
    { 
        String input = "TechHiveOnline";
        System.out.println("The given string: " + input);
        StringReverse obj = new StringReverse(); 
        System.out.print("The reversed string: ");
        obj.reverseString(input); 
          
    }     
}

Output

            The given string: TechHiveOnline
            The reversed string: enilnOeviHhceT

Working of the above code:

1.  Main Method:
      A string input is defined with the value "TechHiveOnline".
      It prints the original string:
      The given string: "TechHiveOnline".
2.  Creating an Object:
      The program creates an object obj of the class StringReverse.
3.  Calling the Recursive Method:
      The method reverseString(input) is called on the object obj, starting the recursive string reversal process.
Example of the Recursive Flow for "TechHiveOnline":
        Initial Call:  reverseString("TechHiveOnline")
        Prints:   e (last character)
        Calls:   reverseString("TechHiveOnlin")
        Prints:   n (last character)
        Calls:   reverseString("TechHiveOnli")
        Prints:   i
        Calls:   reverseString("TechHiveOn")
        Continues...
Eventually, the recursion reaches the base case with a single character, and the full reversed string is printed.
