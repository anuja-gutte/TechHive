Time Complexity

Time complexity is a measure of the amount of time an algorithm takes to complete as a function of the length of the input. It helps in understanding how the algorithm's running time grows with the input size.

Space Complexity

Space complexity refers to the amount of memory space an algorithm uses in relation to the input size. It considers both the space needed for the inputs as well as the auxiliary space required for the algorithm's execution.
