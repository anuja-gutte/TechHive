Queue Implementation Using Arrays

Array Implementation

To implement a queue using a simple array,

  • create an array arr of size n
  • take two variables front and rear which are initialized as 0 and -1 respectively
  • rear is the index up to which the elements are stored in the array including the rear index itself. We mainly add an item by incrementing it.
  • front is the index of the first element of the array. We mainly remove the element at this index in Dequeue operation.
