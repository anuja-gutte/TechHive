Code Implementation of QUEUE - Java

In this section, we will explore the fundamental operations of a queue, including enqueue, dequeue, peek, and print. To help solidify our understanding of these operations, we'll implement a basic queue in Java. This hands-on example will allow you to grasp the core concepts of queue behavior effectively. Furthermore, you are encouraged to try implementing this in other programming languages as well . Doing so will deepen your comprehension of the topic and provide valuable insights into how queues can be handled across different programming environments.

Simple Example : 

        import java.util.*; 
    
        class Queue {
            private static int front, rear, capacity;
            private static int queue[];
        
            Queue(int val)
            {
                front = rear = 0;
                capacity = val;
                queue = new int[capacity];
            }
        
            static void queueEnqueue(int data)
            {
                if (capacity == rear) {
                    System.out.println("Queue is full");
                    return;
                } else {
                    queue[rear] = data;
                    rear++;
                }
                return;
            }
        
            static void queueDequeue()
            {
                if (front == rear) {
                    System.out.println("Queue is empty");
                    return;
                } else {
                    for (int i = 0; i < rear - 1; i++) {
                        queue[i] = queue[i + 1];
                    }
                    if (rear < capacity)
                        queue[rear] = 0;
                    rear--;
                }
                return;
            }
        
            static void queueDisplay()
            {
                int i;
                if (front == rear) {
                    System.out.println("Queue is Empty");
                    return;
                }
                for (i = front; i < rear; i++) {
                    System.out.printf("%d ", queue[i]);
                }
                System.out.println();
                return;
            }
        
            static void queueFront()
            {
                if (front == rear) {
                    System.out.println("Queue is Empty");
                    return;
                }
                System.out.print("Front Element is: "+queue[front]);
                return;
            }
        }
        
        public class StaticQueueinjava {
            public static void main(String[] args)
            {
                Queue q = new Queue(4);
        
                q.queueDisplay();
                q.queueEnqueue(20);
                q.queueEnqueue(30);
                q.queueEnqueue(40);
                q.queueEnqueue(50);
                q.queueDisplay();
                q.queueEnqueue(60);
        
                q.queueDisplay();
        
                q.queueDequeue();
                q.queueDequeue();
                System.out.println("after two node deletion");
        
                q.queueDisplay();
        
                q.queueFront();
            }
        }

Output is : 

        Queue is Empty
        20 30 40 50 
        Queue is full
        20 30 40 50
        after two node deletion
        40 50
        Front Element is: 40

Explanation of code :

The Java program implements a queue using an array with static methods to perform enqueue, dequeue, and display operations. The `Queue` class tracks the front and rear indices to manage the elements, and it limits the queue size based on the given capacity. The program demonstrates the enqueue and dequeue operations, displaying the queue contents and the front element. When the queue is full, it prevents further insertions, and when empty, it prevents removals. The output shows the queue contents at different stages and prints the front element after two dequeue operations.
