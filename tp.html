Arrays

Arrays store elements in contiguous memory locations, allowing indexed access to elements.

Linked Lists

Linked lists consist of nodes where each node contains data and a pointer to the next node.

Trees

Trees are hierarchical data structures with nodes connected by edges. They are used in search operations.

Graphs

Graphs consist of vertices connected by edges, representing relationships between entities.

Heaps

Heaps are complete binary trees where each node is greater (max heap) or smaller (min heap) than its children.

Hashing

Hashing is a technique to map data to a fixed-size table using a hash function for faster access.
