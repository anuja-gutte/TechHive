Linked List Algorithm and Analysis


Problem Statement: A linked list is a linear data structure where each element (node) is a separate object. Each node comprises two fields: data and a reference (or pointer) to the next node in the sequence. Linked lists are dynamic and allow efficient insertion and deletion of elements. Below are the algorithms for inserting and deleting nodes in a singly linked list.

Step-by-Step Algorithm for Insertion at the Beginning:

Step 1:   Start.

Step 2:   Allocate memory for the new node.

Step 3:   Assign data to the new node.

Step 4:   Make the next of the new node point to the current head.

Step 5:   Update the head to point to the new node.

Step 6:   End.

Step-by-Step Algorithm for Deletion from the Beginning:

Step 1:   Start.

Step 2:   If the list is empty, return.

Step 3:   Store the current head node in a temporary variable.

Step 4:   Move the head to point to the next node.

Step 5:   Free the memory of the old head node.

Step 6:   End.

Pseudocode

                
// Node structure
struct Node {
    int data;
    struct Node* next;
};

// Function to insert a new node at the beginning
void insertAtBeginning(struct Node** head, int newData) {
    // 1. Allocate memory for the new node
    struct Node* newNode = (struct Node*)malloc(sizeof(struct Node));

    // 2. Assign data to the new node
    newNode->data = newData;

    // 3. Make next of new node as head
    newNode->next = *head;

    // 4. Move the head to point to the new node
    *head = newNode;
}

// Function to delete a node from the beginning
void deleteAtBeginning(struct Node** head) {
    if (*head == NULL) return; // List is empty

    // 1. Store the current head node in a temp variable
    struct Node* temp = *head;

    // 2. Move head to point to the next node
    *head = (*head)->next;

    // 3. Free memory of the old head
    free(temp);
}

Code Example


    // Singly Linked List Insertion and Deletion in C
    public class LinkedListOperations {
        
        public static void insertAtBeginning(Node** head, int newData) {
            // Allocate memory for the new node
            Node newNode = new Node();
            newNode.data = newData;
            newNode.next = head;
            head = newNode;
        }

        public static void deleteAtBeginning(Node** head) {
            // If the list is empty
            if (head == NULL) return;
            
            Node temp = head;
            head = head.next;
            temp = null;  // Free the old head
        }
    }

Analysis


  1. The insertion and deletion at the beginning of a linked list are very efficient because these operations do not require traversing the list. Both operations are performed in constant time.
  2. For insertion, we allocate memory for a new node, update its next pointer, and adjust the head pointer. This is done in O(1) time.
  3. For deletion, we move the head to the next node and free the memory of the original head, also an O(1) operation.
  4. Therefore, both the time and space complexity for these operations is: O(1).
