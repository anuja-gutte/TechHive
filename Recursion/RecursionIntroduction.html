Recursion

Recursion is a programming technique where a function calls itself directly or indirectly to solve a problem. It is often used when a problem can be divided into smaller subproblems of the same type. In a recursive function, there is usually a base case that stops the recursion and prevents an infinite loop.

How Recursion Works

When a recursive function is called, it breaks down the problem into smaller instances of the same problem.
Each recursive call addresses a subproblem, and this continues until the base case is reached.

The base case provides a simple, non-recursive solution that can be used to solve the smallest instances of the problem.

After reaching the base case, the function returns its result, which is then used to resolve the larger instances of the problem.



Example Code

A classic example of recursion is calculating the factorial of a number. The factorial of a non-negative integer n is the product of all positive integers less than or equal to n. It can be defined recursively as follows: 


            factorial(n) = 1, if n = 0
            factorial(n) = n * factorial(n-1), if n > 0

Examples of Recursion

Some common examples of recursion include:

  • Factorial Calculation: A factorial of a number can be calculated using a recursive function that multiplies the number by the factorial of the number minus one.

  • Fibonacci Sequence: The Fibonacci sequence can be generated by a recursive function where each term is the sum of the two preceding ones.

  • Tree Traversals: Recursive methods are commonly used to traverse trees, such as binary trees, by visiting each node and applying the same procedure to its children.

  • Towers of Hanoi: This classic problem involves moving a stack of disks from one peg to another, following specific rules. It is typically solved using recursion.
