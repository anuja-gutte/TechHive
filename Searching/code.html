1]Linear Search

Linear search is one of the simplest searching algorithms used to find an element in a list or array.
It checks every element in the list sequentially until the desired element is found or the end of the list is reached.

Code:


                    
                    #include 

                        int linearSearch(int arr[], int size, int target) {
                            for (int i = 0; i < size; i++) {
                                if (arr[i] == target) {
                                    return i; // Return the index where the target is found
                                }
                            }
                            return -1; // Return -1 if the target is not found
                        }
                        
                        int main() {
                            int arr[] = {5, 2, 9, 1, 6};
                            int size = sizeof(arr) / sizeof(arr[0]);
                            int target = 9;
                        
                            int result = linearSearch(arr, size, target);
                        
                            if (result != -1) {
                                printf("Element found at index %d\n", result);
                            } else {
                                printf("Element not found in the array\n");
                            }
                        
                            return 0;
                        }

Output:

                        Element found at index 2

Working of the above code:

The linearSearch function goes through each element in the array one by one.
Index 0: 5 (no match).
Index 1: 2 (no match).
Index 2: 9 (match found).
Since the target 9 is found at index 2, the function returns 2 and the search ends.

2]Binary Search

Binary search is a much more efficient algorithm than linear search but works only on sorted arrays.
It repeatedly divides the search interval in half,
eliminating half of the elements at each step until the target value is found or the interval is empty.

Code

                    
                    #include 

                        int binarySearch(int arr[], int size, int target) {
                            int left = 0, right = size - 1;
                        
                            while (left <= right) {
                                int mid = left + (right - left) / 2;
                        
                                // Check if the target is present at mid
                                if (arr[mid] == target)
                                    return mid;
                        
                                // If target is greater, ignore the left half
                                if (arr[mid] < target)
                                    left = mid + 1;
                                // If target is smaller, ignore the right half
                                else
                                    right = mid - 1;
                            }
                        
                            // Target is not present in the array
                            return -1;
                        }
                        
                        int main() {
                            int arr[] = {2, 3, 4, 10, 40};
                            int size = sizeof(arr) / sizeof(arr[0]);
                            int target = 10;
                        
                            int result = binarySearch(arr, size, target);
                            if (result != -1)
                                printf("Element found at index %d\n", result);
                            else
                                printf("Element not found\n");
                        
                            return 0;
                        }

Output:

                        Element found at index 3

Working of the above code:

1.Initial Setup: The binarySearch function starts by setting left to the first index (0)
and right to the last index (size - 1 = 4).
2.The array is divided into halves in each step:
First iteration:
The midpoint mid is calculated as 2 (using (left + right) / 2).
The element at mid is 4, which is less than the target 10,
so the search continues in the right half (i.e., from index 3 to 4).
Second iteration:
Now left is updated to 3, and right remains 4. The new midpoint mid is 3.
The element at mid is 10, which is the target, so the function returns index 3.
3.The function ends successfully when the target is found.
