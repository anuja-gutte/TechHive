Arrays

Arrays are a collection of elements, stored at contiguous memory locations. They are widely used in programming as a basic data structure to store and access elements efficiently.

Array Algorithms

Some important algorithms related to arrays include:

  • Searching (Linear Search, Binary Search)
  • Sorting (Bubble Sort, Quick Sort, Merge Sort)
  • Array Rotation
  • Subarray Sum

Example Code


  int array[5] = {1, 2, 3, 4, 5};
  for (int i = 0; i < 5; i++) {
    printf("%d ", array[i]);
  }
